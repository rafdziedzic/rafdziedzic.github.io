- Spring: SpringBoot, SpringData, SpringSecurity, Thymeleaf
- Warstwe Front-end wykonano za pomocą HTML,CSS,Bootstrap
- Wykorzystano również aplikacje wspomagającą budowe projetu: Maven

- Apache Maven
- Apache Tomcat
- Warstwę Front-end zbudowano za pomocą: HTML, CSS, Bootstrap
